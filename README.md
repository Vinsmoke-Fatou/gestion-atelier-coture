# Tailoring Workshop Management System – C++ Object-Oriented Application

## Overview
This desktop console application is a specialized management platform tailored for custom tailoring workshops and fashion design businesses. Developed using C++, this project leverages advanced Object-Oriented Programming (OOP) principles to digitize, automate, and manage client measurements, material stocks, and garment production workflows.

## Features

- Client & Anatomical Measurement Registry:
  _Centralized registry for managing customer contact profiles and order history.
  _Detailed digital storage of custom anatomical measurements required for precision dressmaking.

- Order Processing & Status Tracking:
  _End-to-end processing of tailoring orders from initial client booking to final assembly.
  _Real-time status updates across production stages (e.g., cutting, sewing, ready for collection) to keep workflows organized.

- Material & Fabric Stock Controls:
  _Real-time tracking of available fabrics, trims, and tailoring materials in storage.
  _Logic-driven quantity checks to align available workshop materials with incoming order requirements.

- Object-Oriented Software Design:
  _Robust class design utilizing encapsulation to safeguard delicate client and order data.
  _Clean data structures designed for memory management and efficient console-based operations.
